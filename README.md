# EVELYN WEB - 個人作品集網站

這是一個使用 Vue.3 建立的個人作品集網站，展示平面設計、UI/UX 設計和影片編輯等作品。

## 🎨 專案特色

- **Vue 3 框架**: 使用最新的 Vue 3 Composition API
- **輪播功能**: 自動輪播廣告圖片展示
- **作品展示**: 互動式作品集瀏覽體驗
- **GitHub Pages 部署**: 免費託管靜態網站

## 🌐 線上展示

網站連結:https://evelyn401.github.io/WEB/

## 📁 專案結構

```
WEB/
├── index.htm           # 主頁面
├── css/                # 樣式表資料夾
│   ├── styles.css      # 主要樣式
│   └── web/
│       ├── indexmain.css
│       ├── header.css
│       └── footer.css
├── vue/                # Vue 程式碼
│   └── vue.js          # Vue 應用程式邏輯
├── images/             # 圖片資源
│   ├── CarouselXXL/    # 輪播圖片
│   ├── icons/          # 技能圖示
│   ├── work/           # 作品圖片
│   └── background/     # 背景圖片
└── README.md           # 專案說明文件
```

## 🛠️ 技術棧

- **HTML5**: 語意化標籤
- **CSS3**: Flexbox、Grid 佈局、RWD 響應式設計
- **JavaScript ES6+**: 模組化程式碼
- **Vue 3**: 前端框架，使用 ESM 模組載入
- **GitHub Pages**: 網站託管服務


## 📝 主要功能

### 1. 輪播廣告 (Carousel)
- 自動輪播，每 3 秒切換一次
- 支援手動切換 (上一張/下一張按鈕)
- 使用 Vue 響應式資料控制

### 2. 技能展示 (Skills)
- 三大技能類別：平面設計、UI/UX 設計、影片編輯
- 卡片式佈局展示

### 3. 作品集 (Works)
- 九件作品展示
- 點擊可放大檢視
- 使用背景圖片和遮罩效果

### 4. 聯絡表單 (Contact)
- 使用者資訊收集
- 表單驗證功能
